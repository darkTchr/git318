git init 初始化仓库 git add 文件名 将文件添加到暂存区 git commit -m '记录的内容' 告诉git把文件提交到仓库 (第一次提交不会很顺利 , 会保存 ,需要补全个人信息 : 名字 和 邮箱) git config --global user.email "xxx@qq.com" git config --global user.name "jack" git log 查看记录的信息 git status 查看是否还有其他文件没有提交 (nothing to commit) ------注意-------- git add . add和.之间有空格 , 将执行命令所在的目录位置以及后代目录下的文件add git add all 无论在哪个目录下执行都会提交相应的文件 git bash 想要执行复制粘贴,难受,enter/右键->粘贴 用户名和邮箱 至设置一次就行 ----------------- 版本穿梭 git reset --hard 版本号
